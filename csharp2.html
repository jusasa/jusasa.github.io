언어 주요 특징

객체지향(OOP) 문법 및 활용

클래스, 속성, 인덱서, 그리고 다형성의 핵심인 인터페이스를 배웁니다.

핵심 문법 요소

  • 클래스(Class): 객체의 설계도 (예: Player, Monster)
  • 속성(Property): 변수처럼 보이지만 내부적으로 get/set 메서드로 동작하여 데이터 캡슐화 및 유효성 검사 수행.
  • 인덱서(Indexer): 객체를 배열처럼 `[]`로 접근하게 해주는 문법.
  • 인터페이스(Interface): 다중 상속이 가능하며, 구체적 구현 없이 '기능의 약속'만 정의하여 다형성을 제공.

실제 사용 예시 (Real-world Use Cases)

⚔️

RPG 게임 전투 시스템

`Player`와 `Monster` 클래스를 정의. 각 객체는 체력(HP) 속성을 가지며, 무기 인터페이스(`IWeapon`)를 통해 다형성(검, 총 등)을 구현하여 공격(`Attack()`) 메서드 실행.

🏠

스마트홈 기기 제어

`IPower`(전원 켜기/끄기), `IInternet`(네트워크 연결) 인터페이스를 분리. 스마트TV, 밥솥 클래스가 필요한 규약만 다중 상속받아 각각의 방식대로 기능 구현.

다차원 배열과 가변 배열

같은 자료형의 데이터를 묶어서 관리하는 구조. 차원(Dimension)에 따라 활용처가 다릅니다.

1차원 배열
int[] scores = new int[5];

단일 목록형 데이터를 저장합니다. 반복문(`for`)과 함께 사용되어 효율성을 극대화합니다.

실 사용 예시: 학생 성적 관리 5명의 학생 점수를 하나의 배열에 담고, 반복문으로 총점, 평균, 최고/최저점을 계산.
2차원 배열
int[,] lotto = new int[5, 6];

행과 열로 구성된 표(Table/Grid) 형태의 데이터를 저장합니다.

실 사용 예시: 로또 번호 생성기 5줄(행)의 로또를 구매할 때, 각 줄마다 6개(열)의 당첨 번호를 중복 없이 저장하고 출력.
가변 배열 (Jagged)
int[][] seats = new int[3][];

배열 안의 배열. 각 행마다 열의 길이가 다를 수 있어 메모리를 유연하게 사용합니다.

실 사용 예시: 영화관 좌석 예약 1관(10석), 2관(8석), 3관(12석)처럼 상영관마다 다른 좌석 수를 가변 배열로 최적화하여 구현.

예외 처리 및 파일/데이터 관리

프로그램의 안정성을 높이고 데이터를 영구적으로 보존하는 기법입니다.

예외 처리 (try-catch-finally)

실행 중 발생하는 오류로 인한 프로그램의 강제 종료를 방지합니다.

try {
  // 오류 발생 가능 코드 (ex: 0으로 나누기)
} catch (DivideByZeroException) {
  // 예외 발생 시 처리 로직
} finally {
  // 항상 실행 (ex: ATM 카드 반환, 자원 해제)
}

파일 입출력 & JSON 직렬화

메모리의 휘발성 데이터를 `FileStream`, `StreamWriter`를 통해 하드디스크에 저장합니다.

실 사용 예시: 학생 데이터 저장/불러오기
  • 텍스트 기반(TXT): 데이터를 구분자(',')로 저장 후 분리(`Split`)하여 읽기.
  • 직렬화(JSON): `Student` 객체 자체를 표준 텍스트 포맷(JSON)으로 변환하여 통신/저장. JsonSerializer.Serialize(student) 활용.
