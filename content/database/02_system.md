### ⚙️ CH02. 데이터베이스 시스템 (DBS)

#### 데이터베이스 언어

- DDL (데이터 정의어): DB 구조, 스키마 정의 (`CREATE`, `ALTER`, `DROP`)

- DML (데이터 조작어): 데이터 검색, 삽입, 수정, 삭제 (`SELECT`, `INSERT`, `UPDATE`, `DELETE`)

- DCL (데이터 제어어): 보안, 권한 제어 (`GRANT`, `REVOKE`)

#### ANSI/SPARC 3단계 스키마

- 외부 스키마: 개별 사용자 관점 (뷰)

- 개념 스키마: 조직 전체 관점 (통합 논리 구조)

- 내부 스키마: 물리적 저장 장치 관점

>💡 핵심 목적: 스키마 간 사상을 통해 응용 프로그램과 구조를 분리하는 데이터 독립성을 제공하기 위함입니다.
