### ⚙️ 핵심 문법 및 빌드 프로세스

프로그램이 어떻게 작성되고 실행되는지 근본적인 과정을 이해하는 섹션입니다. 사람이 작성한 텍스트 코드가 기계가 이해할 수 있는 실행 파일로 변환되는 전체 흐름(빌드, 컴파일, 링크)을 확인할 수 있습니다.

#### C# 프로그램 빌드 순서

1. 소스 파일 (*.cs): 프로그래머가 작성한 C# 코드

2. 컴파일 (기계어 변환): 소스 코드를 컴퓨터가 이해할 수 있는 언어로 변환

3. 오브젝트 파일 (중간 산출물): 컴파일의 결과물

4. 링크 (라이브러리 결합): 여러 오브젝트 파일과 필요한 라이브러리를 하나로 연결

5. 실행 파일 (*.exe): 최종적으로 사용자가 실행할 수 있는 프로그램
