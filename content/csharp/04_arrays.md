### 📦 데이터 구조: 배열(Array) 분석

동일한 자료형의 데이터를 체계적으로 관리하는 배열의 종류와 특징을 비교합니다.

#### 1차원 배열 (1D Array)

가장 기본적인 형태. 하나의 선(Line) 구조로 데이터를 저장합니다.

```cs
int[] numbers = new int[5];
```


#### 2차원 배열 (2D Array)

행(Row)과 열(Column)로 구성된 표(Table) 구조. 모든 행의 길이가 동일합니다.
```cs
int[,] matrix = new int[2, 3];
```

#### 가변 배열 (Jagged Array)

'배열의 배열'. 각 행의 열 크기가 다를 수 있어 동적인 메모리 관리에 유리합니다. 구조적 복잡도는 높지만 불필요한 메모리 할당을 줄여 유연성을 보장합니다.
````cs
int[][] jagged = new int[3][];
````
