### 15. 입출력 스트림

#### 스트림이란?

자료가 오고가는 통로

#### 스트림의 종류

* 입력 스트림
* 출력 스트림

> 스트림은 단방향으로 동작하기에 입력 출력이 동시에 불가하다.

#### 바이트단위와 문자단위

* 바이트 스트림
    -   BufferedInputStream
    -   BufferedOutputStream
    -   데이터를 바이트 단위로 읽고 쓴다. 이미지, 오디오 파일, 동영상 파일 등을 다룰 때 사용
* 문자 스트림
    -   BufferedReader
    -   BufferedWriter
    -   텍스트 데이터를 문자 단위로 읽고 쓴다. 텍스트 파일, JSON 파일, XML 파일 등을 다룰 때 사용

    #### 기반 스트림과 보조스트림

    * 기반스트림
        * 직접 파일을 읽거나 쓰는놈
    * 보조스트림
        * 기반스트림을 감싸서 성능을 향상시키거나 추가 기능을 제공하는놈

### 표준 입출력

#### 
* `System.in`: 표준 입력 스트림 (키보드 입력)
* `System.out`: 표준 출력 스트림 (화면 출력)
* `System.err`: 표준 에러 스트림 (화면 출력)
