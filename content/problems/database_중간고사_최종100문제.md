## Part 1. 데이터베이스 이론 (40%, 40문제)

### [1장. 데이터베이스 개념]

### 1. [주관식] 데이터베이스의 4가지 주요 정의 중, 여러 곳에서 필요한 데이터를 하나로 모았다는 의미를 가지는 데이터는 무엇인가?

### 2. [주관식] 데이터베이스의 정의 중, 특정 조직의 목적을 위해 반드시 필요한 필수 데이터의 모임을 뜻하는 용어는 무엇인가?

### 3. [주관식] 데이터베이스의 특성 중, 다수의 사용자가 동시에 접근하여 사용할 수 있는 성질을 무엇이라 하는가?

### 4. [주관식] 데이터베이스의 특성 중, 수많은 사용자의 요구에 대해 보통 수 초 안에 응답해야 하는 성질을 무엇이라 하는가?

### 5. [주관식] 데이터베이스의 데이터를 참조할 때 물리적인 주소가 아닌 데이터가 가진 '값' 자체로 참조하는 특징을 뜻하는 용어는 무엇인가?

### 6. [주관식] 데이터베이스는 현실 세계가 변화하면 이를 지속적으로 반영해야 한다. 이를 무엇이라 하는가?

### 7. [주관식] 기존 파일 시스템의 가장 큰 단점 두 가지는 데이터의 '종속성'과 무엇인가?

### [2장. 데이터베이스 시스템]

### 8. [주관식] 3단계 데이터베이스 구조에서 데이터베이스의 전체적인 논리적 구조를 정의하며, DBA가 주로 관리하는 스키마는 무엇인가?

### 9. [주관식] 3단계 데이터베이스 구조에서 일반 사용자나 응용 프로그래머가 접근하는 사용자 관점의 스키마는 무엇인가?

### 10. [주관식] 물리적 저장 장치가 재구성되더라도 응용 프로그램이나 논리적 구조가 영향을 받지 않는 성질을 'OO적 데이터 독립성'이라 한다. 빈칸은?

### 11. [주관식] 데이터베이스에 저장된 모든 부가 정보, 즉 스키마, 제약 조건 등의 '데이터에 관한 데이터(메타 데이터)'를 저장하는 저장소를 무엇이라 하는가?

### 12. [주관식] 데이터베이스 사용자의 종류 중, 데이터베이스 설계, 관리 및 보안, 권한 부여 등 전반적인 책임을 지는 그룹을 무엇이라 하는가?

### 13. [주관식] 데이터베이스 언어 중, 데이터의 검색, 삽입, 수정, 삭제를 위해 응용 프로그래머나 일반 사용자가 주로 사용하는 언어는 무엇인가? (예: DML, DDL)

### [3장. 관계형 데이터 모델]

### 14. [주관식] 데이터 모델의 3요소는 데이터를 표현하는 '데이터 구조', 데이터를 처리하는 '연산', 그리고 허용되는 행위의 제한을 두는 'OO OO'이다.

### 15. [주관식] 관계형 모델에서 테이블의 행(Row)을 의미하는 용어는 무엇인가?

### 16. [주관식] 관계형 모델에서 테이블의 열(Column)을 의미하는 용어는 무엇인가?

### 17. [주관식] 하나의 속성(Attribute)이 가질 수 있는 원자값(Atomic Value)들의 집합을 무엇이라 하는가?

### 18. [주관식] 릴레이션에 존재하는 튜플의 총 개수를 무엇이라 하는가?

### 19. [주관식] 릴레이션에 존재하는 속성의 총 개수(차수)를 무엇이라 하는가?

### 20. [주관식] 후보키(Candidate Key)가 되기 위해 만족해야 하는 두 가지 조건은 '유일성'과 무엇인가?

### 21. [주관식] 데이터베이스 설계자가 후보키 중에서 튜플을 유일하게 식별하기 위해 특별히 선택한 키를 무엇이라 하는가?

### 22. [주관식] "기본키는 널(NULL) 값을 가질 수 없다"는 무결성 제약조건의 명칭은 무엇인가?

### 23. [주관식] 외래키(Foreign Key)의 값은 참조하는 릴레이션의 기본키 값이거나 NULL이어야 한다는 제약조건은 무엇인가?

### 24. [객관식] (O/X) 릴레이션에서 튜플들의 순서는 의미가 없으며, 속성들의 순서 또한 의미가 없다.

### [4장. 관계 대수]

### 25. [주관식] 릴레이션을 처리하기 위해 코드(Codd)가 정의한 연산으로, 원하는 결과를 얻기 위한 처리 '과정'을 명시하는 절차적 언어는 무엇인가?

### 26. [주관식] 릴레이션에서 특정 조건을 만족하는 튜플(행)들만 추출하는 단항 연산자는 무엇인가? (기호: $\sigma$)

### 27. [주관식] 릴레이션에서 주어진 특정 속성(열)들의 값만 추출하여 부분 릴레이션을 만드는 연산자는 무엇인가? (기호: $\pi$)

### 28. [주관식] 공통 속성을 기준으로 두 릴레이션을 결합하여 새로운 릴레이션을 만드는 연산을 무엇이라 하는가? (기호: $\bowtie$)

### 29. [주관식] 두 릴레이션이 합집합, 교집합, 차집합 연산을 수행하기 위해 속성의 개수와 도메인이 일치해야 한다는 조건을 무엇이라 하는가? (OO 양립성)

### 30. [주관식] 조건식 없이 두 릴레이션의 모든 튜플을 각각 결합하여 결과를 생성하는 연산으로, 결과 튜플 수는 두 릴레이션 튜플 수의 곱이 되는 연산은?

### 31. [주관식] SQL의 명령어 중 CREATE, ALTER, DROP은 어느 부속 언어에 속하는가? (영문 약어 3글자)

### 32. [주관식] 검색 결과를 특정 기준에 따라 정렬하여 보여주기 위해 사용하는 SELECT 문의 절(Clause)은 무엇인가?

### 33. [주관식] SELECT 검색 결과에서 중복되는 튜플을 한 번만 출력하도록 지정하는 키워드는 무엇인가?

### 34. [주관식] 데이터의 패턴을 검색할 때 사용하며, '모든 문자(길이 무관)'를 뜻하는 % 와 '단일 문자'를 뜻하는 _ 기호를 사용하는 연산자는 무엇인가?

### 35. [주관식] SELECT 명령문에서 그룹 함수를 적용하기 위해 특정 속성을 기준으로 그룹화할 때 사용하는 절은 무엇인가?

### 36. [주관식] 동등 조인(Equi Join)의 결과 릴레이션에서 중복된 속성을 제거하여 표시하는 조인을 무엇이라 하는가?

### 37. [주관식] 두 테이블을 조인할 때, 조인 조건에 일치하지 않아 참여하지 못하는 튜플들까지 결과에 포함시키는 조인은 무엇인가?

### 38. [주관식] 테이블에 새로운 튜플을 추가할 때 사용하는 DML 명령어는 무엇인가?

### 39. [주관식] 테이블의 기존 데이터를 수정할 때 사용하는 DML 명령어는 무엇인가?

### 40. [객관식] (O/X) UPDATE 문이나 DELETE 문에서 WHERE 절을 생략하면 테이블의 모든 행이 한꺼번에 수정되거나 삭제된다.---
